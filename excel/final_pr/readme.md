# Sales Data Analysis & Interactive Dashboard

## Project Overview

This project is a comprehensive data analysis and visualization tool built in Google Sheets. It transforms raw transactional data into actionable insights through advanced formulas, pivot tables, and an interactive dashboard. The project was designed to demonstrate proficiency in data structuring, analytical modeling, and reporting.

## Key Features

### Dynamic Revenue Tracking
Automatically calculates revenue per transaction based on quantity and unit price.

### Advanced Data Analysis

#### Date & Time Analysis
Uses TODAY, NOW, EOMONTH, and DATEDIF for real-time reporting.

#### Filtering
Implements FILTER functions to isolate specific product categories (e.g., Electronics).

#### Customer Segmentation
Identifies "High-Value Customers" based on total revenue contributions.

#### String Manipulation
Uses text functions to generate category abbreviations and conduct product-level analysis.

#### List Matching
Employs comparison formulas to find matching names across different datasets.

## Interactive Dashboard

### KPI Scorecards
Displays Total Revenue and Total Orders at a glance.

### Visual Reports
Includes Pie, Line, and Bar charts for revenue by category, trends over time, and product sales.

### User Interactivity
Features a Slicer to allow users to filter the entire dashboard by product category.

## Spreadsheet Structure

### Instructions
A guide on how the spreadsheet is organized and how to interact with the data.

### Raw Data (Hidden)
The foundational dataset containing over 200 transaction records.

### Analysis
The "engine room" where complex formulas process the raw data into summary tables.

### Visualizations
Houses the pivot tables and source charts used to build the final report.

### Dashboard
The primary user interface for viewing high-level metrics and interacting with visualizations.
