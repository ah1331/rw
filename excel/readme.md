# PR.1 Fundamental Booster — Excel Practice Workbook

An Excel workbook for practicing core spreadsheet formulas using real-world datasets (Students, Sales, Employees).

---

## Sheets Overview

| Sheet | Purpose |
|-------|---------|
| `Sheet1` | Source data — Students, Sales, and Employee records |
| `Sheet2` | Empty working sheet |
| `Main Tasks` | 6 hands-on tasks to complete using formulas |
| `Solutions` | Reference answers for all tasks |

---

## Topics Covered

- Relative & Absolute Cell References, Formatting, Data Input
- IF Formulas, Nested IFs, IF with AND/OR
- COUNTIFS, SUMIFS, AVERAGEIFS
- Lookup Functions — VLOOKUP, XLOOKUP, XMATCH
- INDEX/MATCH for dynamic searches
- TEXT Functions for string manipulation
- INDIRECT and OFFSET for dynamic references
- Date/Time, Math Functions, FILTER function

---

## Tasks

1. Apply IF formulas to classify student grades (A/B/C/F)
2. Use COUNTIFS to count students scoring above 60
3. Use VLOOKUP to find a product price from sales data
4. Apply XLOOKUP to fetch employee salaries dynamically
5. Format and analyze employee joining dates
6. Use FILTER to extract students with grades above 80

---

## Deliverable

Complete all 6 tasks in `Main Tasks` sheet using Excel formulas that reference source data from `Sheet1`. Check answers in the `Solutions` sheet.
