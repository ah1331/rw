# PR.2 Analyzer

Excel-based data analysis project on 200 customer sales transactions.

## Dataset Columns
`Customer_ID` · `Customer_Name` · `Region` · `Product_Category` · `Sales` · `Quantity` · `Discount` · `Order_Date` · `Profit`

## Tasks Covered

| # | Task | Tool/Feature |
|---|------|-------------|
| 1 | Top 10 customers by total purchase | Conditional Formatting |
| 2 | Discount impact on profit | What-If Analysis |
| 3 | Profit vs Sales relationship | Linear Regression |
| 4 | Full statistical summary | Descriptive Statistics |
| 5 | Monthly sales growth | Custom Arrows (▲ ▼) |
| 6 | Record entry timestamps | NOW() Function |
| 7 | Customer segmentation | Index / Match / GroupBy |
| 8 | Sales & profit breakdown | Pivot Table |
| 9 | Visual KPIs | Bar · Line · Pie Charts |
| 10 | Business summary | Dashboard Insights |

## Key Findings
- **Top Region:** West ($44,083)
- **Top Product:** Books ($47,002)
- **Top Customer:** CUST015 ($11,611)
- **Profit Margin:** ~35%
- **Regression R²:** 0.60 — Sales strongly predicts Profit

## File
`PR2_Analyzer_Complete.xlsx` — all 10 tasks answered in one sheet, no color formatting.
